Install:
Node 10.16

Built with:
- Backend: NodeJS
- Frontend: Angular

Running backend:
- cd backend
- npm install
- npm start

Running frontend:
- cd frontend
- npm install
- npm start

- Charts: chart.js => https://github.com/chartjs/Chart.js
